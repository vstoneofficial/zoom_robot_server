# 概要  
 Vstone株式会社のメガローバー・メカナムローバー(研究開発用大型台車ロボット)を利用して、テレイグジスタンスロボットを構築するプロジェクトのサンプルソースです。  

 本プロジェクトは、台車ロボットのPC側(Ubuntu)のプロジェクトになります。
